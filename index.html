mfreq

FTN file request application

Tools:
Navigation: Home | mfreq

The mfreq suite of tools is for FTN style file requesting. It consists of a file index generator, a filelist generator, and an SRIF compatible file request handler.

  • The master branch in the repository contains the code from the mfreq application itself.
  • The debian branch in the repository is used for the development of the debian packaging directory for mfreq. It includes the mfreq code merged from the master branch.
  • The precise branch in the repository is used for the development of the Ubuntu v12.04 (precise) packaging directory for mfreq. It includes the mfreq code merged from the master branch.
